# Raspberry-Pi-Starter
Tired of running a bunch of the same commands every time I get a new Pi going.  This is just something simple so I can run one line and be done with all the initial stuff I usually do.

Use:

```
curl -fsSL https://tinyurl.com/startrpi | sh
```

Or:

```
curl -fsSL https://wesleymcginn.github.io/Raspberry-Pi-Starter/start.sh
```
